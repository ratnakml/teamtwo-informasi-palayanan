Format Reset Password

Kirim format berikut untuk melakukan reset:

ASK#RESET#NIM#PasswordBaru

Contoh:

ASK#RESET#1234567890#PasswordBaru123

Setelah pesan dikirim, sistem akan memproses permintaan Anda dan password akan diperbarui otomatis.

Pastikan:
• NIM dituliskan dengan benar
• Password baru mudah diingat namun tetap aman
• Tidak membagikan format reset kepada orang lain
